## 🎨 **DESIGN SYSTEM**

### **Colors:**
```css
Primary: Black (#000000)
Secondary: White (#FFFFFF)
Accent: Amber (#f59e0b, #d97706)
Success: Green (#16a34a, #15803d)
Danger: Red (#dc2626, #ef4444)
Background: Gray-50 (#f9fafb)
```

### **Typography:**
```css
Headings: 'Playfair Display' (serif)
Body: 'Inter' (sans-serif)
Sizes: 5xl → 7xl → 8xl (responsive)
Weights: 300, 400, 600, 700, 800
```

### **Spacing:**
- Section padding: py-16 → py-20
- Card padding: p-5 → p-8
- Grid gaps: gap-6 → gap-8
- Premium spacing throughout

### **Shadows:**
```css
Card: shadow-lg → shadow-2xl on hover
Premium: shadow-xl with custom colors
Floating: shadow-2xl
```
